class src.scheduler.task.Task(function_name: str, args: dict[Any, Any] | None = None)[source]

A class representing GUTS scheduler tasks.

Tasks are units of work that workers can execute. They can be serialized, stored in the scheduler database and retrieved later for execution.

The current implementation relies on an in-memory function register to map function names stored in the tasks to Python concrete functions. A disk-based function register might be better suited in this framework.
